## ✅ 修复的问题

### 问题 1: CartItemDTO 缺少 shopId 字段

**现象**: 前端提交订单时报 500 错误  
**原因**: `CartItemDTO` 没有 `shopId` 字段，导致前端无法获取店铺 ID  
**修复**:
- `CartItemDTO.java` - 添加 `shopId` 字段
- `CartItemServiceImpl.java` - 设置 `shopId = dish.getShopId()`

### 问题 2: OrderService 缺少 List 导入

**现象**: 编译失败  
**错误**: `cannot find symbol: class List`  
**修复**: `OrderService.java` - 添加 `import java.util.List;`

### 问题 3: 订单列表接口 404

**现象**: `GET /order/list` 返回 404  
**原因**: 编译失败导致 Controller 未正确加载  
**修复**: 修复编译错误后重新编译
