# 订单详情修复说明

**修复时间**: 2026-03-25 17:28  
**问题**: 订单详情页面没有关联到菜品信息，order_detail 表没有数据

---

## 🐛 问题原因

1. **创建订单时未插入订单明细** - OrderServiceImpl 只创建了订单主表数据，没有插入 order_detail 表
2. **订单详情接口未返回明细** - getOrderDetail 接口只返回订单主表信息
3. **前端未显示菜品列表** - 订单详情页缺少商品清单展示

**新增方法**:
```java
@Override
public OrderDetailDTO getOrderDetailWithDetails(Long orderId) {
    // 查询订单
    Orders order = this.getById(orderId);
    
    // 查询订单明细
    LambdaQueryWrapper<OrderDetail> queryWrapper = new LambdaQueryWrapper<>();
    queryWrapper.eq(OrderDetail::getOrderId, orderId);
    List<OrderDetail> details = orderDetailMapper.selectList(queryWrapper);
    
    // 组装 DTO
    OrderDetailDTO dto = new OrderDetailDTO();
    dto.setOrder(order);
    dto.setOrderDetails(details);
    
    return dto;
}
```
